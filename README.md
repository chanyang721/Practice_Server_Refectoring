## 구현시 선택한 설계 방향
* 기술 스택 선정 
그동안 미뤄왔던 TS를 시작해야겠다고 생각하여 TS + Express로 기본적인 서버를 구축할 예정이다.
필수 스택중 Jest, TypeDI, Joi는 처음 시도하는 스택들이기 때문에 공부가 필요해 보인다.
